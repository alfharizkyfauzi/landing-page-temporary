About Course

Data Science Master Class course divided into two programs: Intermediate Level and Beginner Level, this course aims to introduce students to a blend of Data Science concepts and technologies in order to enable them to work with everyday Data related issues analytically. Students will gain experience of the entire Data Science supply chain, namely Data collection, processing, analysis, and visualization, using an innovative hands-on practical Data Science Group project.

Designed For

Everyone who doesn’t have any background in IT (Need to join our Fundamental Class first)

You Will Learn

  • Python Programming
    • In this module, student would learn about the Python Programming fundamental and how to use the basic usage of Python language.
    • Data Types : numbers, strings, lists, dictionaries, Boolean
    • Comparison Operators
    • If, else if, and else statements
    • For loops, while loops,
    • Functions, lambda expressions & Algorithm Training
  • Data Science with Python
    • Personal Data Science Project Presentation that student create and presented in front of Trainer and/or Expert. The project should address a business problem and try to solve this problem with data.
  • Data Analysis
    • In this module, student focus on learning how to using various Python data package, learning basic data exploration, analysis theory, and basic statistic.
    • Numpy
    • Pandas
    • Visualization Package (Matplotlib, Seaborn)
    • Statistic
    • SQL Language
    • Exploratory Data Analysis (EDA)
  • Machine Learning
    • In this module, student would focus on the Data Scientist activity and learning the basic concept of Machine Learning algorithm, machine learning modelling, and machine learning evaluation.
    • Supervised Analysis (Regression, Classification)
    • Unsupervised Analysis (Clustering)
    • Model Tuning and Evaluation
    • Case Study: Develop ML for business case
  • Experiments and Causal Inference
    • In this module, student would focus on design practical experiments to gain insights, and learn to deal with external factors and events outside your control.
  • Data Science to Drive Business Value
    • In this module, student would focus on learn how to put the tools and techniques you have learned to the service of business growth and value generation.
  • Final Project
    • Personal Data Science Project Presentation that student create and presented in front of Trainer and/or Expert. The project should address a business problem and try to solve this problem with data.
